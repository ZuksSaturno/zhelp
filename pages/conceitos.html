1. List-style-position

Esse prompt posibilita a mudança de local da bolinha ou do simbolo selecionado da lista seja ela <UL>ou <OL>. 

  <!DOCTYPE Css>

 object{
    list-style-position: inside;
 }

2. Vertical-alignt

Esse prompt faz a mudança de posicionamento do obejeto em um eixo vertical mas não é sempre que ela funciona. 

  <!DOCTYPE Css>
        
 object{
     Vertical-alignt: 200px;       
 }

3. Letter-spacing

Esse prompt posibilita a manipulação do espaçamento entre as letras umas com as outras. 

  <!DOCTYPE Css>
        
 object{
    letter-spacing: 1px;
 }

4. Line-height

Esse prompt posibilita fazer modificação no espaçamento entre as linhas de textos. 

  <!DOCTYPE Css>
        
 object{
     line-height: 40px;
 }

5. Align-items

Esse prompt faz com que o conteudo de certo obejeto fique centralizado . 

  <!DOCTYPE Css>
        
 object{
    align-items: center;
 }

6. list-style-type

Esse prompt faz você conseguir mudar o tipo do point da sua lista. 

  <!DOCTYPE Css>
        
 object{
    list-style-type: '\1F4C1\00a0\00a0';
 }

7. Content-visibility

Esse prompt possibilita deixar oculto o conteudo que esat em algum objeto 

  <!DOCTYPE Css>
        
 object{
    content-visibility: inherit;
 }

8. --variavel:

Esse prompt cria variaveis dentro da tag :root{} 

  <!DOCTYPE Css>
        
 :root{
    --fonttxt:'Quicksand, sans-serif';
 }
