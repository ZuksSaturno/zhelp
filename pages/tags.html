1. <a href ". ">

Tag de link ou seja, ela transforma tudo em um link que quando ativado te redireciona para outro lugar externo a página ou da mesma page. 

  <!DOCTYPE html>
 <html>
  <body>

    <a href="https//link-foda"> Aperte aqui para ser redirecionado </a>
    <a href="https//link-foda"><img src="foto.png" alt="foto"></a>
    <a href="https//link-foda" target="_blank"> Aperte aqui </a>

  </body>
 </html>

2. <Code>

Nessa TAG você deixa o código juntamente com o pré para ficar igual esteticamente a um código de programação.o code deixa a font diferente 

  <!DOCTYPE html>
 <html>
  <body>
    <p>    
      <code>
            Nessa TAG você deixa o código juntamente com o pré para ficar igual
            esteticamente a um código de programação.o code deixa a font diferente
      </code> 
   </p>
  </body>
 </html>

3. <p>

É uma TAG de texto que se baseia em criar uma área block-level designada a ser um paragrafo. 

  <!DOCTYPE html>
 <html>
  <body>
   <p>É uma TAG de texto que se baseia em criar uma área block-level designada
 a  ser um paragrafo.</p>
  </body>
 </html>

4. <h1> até <h7>

São tags de texto sendo elas focadas em títulos sendo h1(negrito) h7(light). 

  <!DOCTYPE html>
 <html>
  <body>
    <h1> TITULO FODA APENAS </h1>
  </body>
 </html>

5. <hr>

A tag hr cria uma linha de divisão que ocupa todo o body width. 

  <!DOCTYPE html>
 <html>
  <body>
    <hr>
  </body>
 </html>

6. <br>

Essa TAG faz uma quebra de linha no html ou seja após usar-la o próximo conteúdo ficará na linha de baixo 

  <!DOCTYPE html>
 <html>
  <body>
    <br>
  </body>
 </html>

7. <em>

Essa TAG é de estilo e server pra fazer um css inline, ela deixa o texto em itálico. 

  <!DOCTYPE html>
 <html>
  <body>
    <em> texto em italico </em>
  </body>
 </html>

8. <strong>

Tag de estilo que deixa o texto em negrito. 

  <!DOCTYPE html>
 <html>
  <body>
    <strong> texto em negrito </strong>
  </body>
 </html>

9. <ol>

Essa TAG faz uma lista ordenada tanto por números, símbolo, números romanos. 

  <!DOCTYPE html>
 <html>
  <body>
    <ol>
        <li> Somar </li>
        <li> Dividir </li>
        <li> Raiz </li>
        <li> Equação </li>
    </ol>
  </body>
 </html>

10. <ul>

Essa TAG cria uma lista desordenada ou seja apenas possui pontos de marcações que podem ser modificados por emojis e etc. 

  <!DOCTYPE html>
 <html>
  <body>
   <ul>
        <li> vaca </li>
        <li> boi </li>
        <li> cavalo </li>
        <li> tatu </li>
   </ul>
  </body>
 </html>

11. <li>

Essa TAG são os elementos de cada lista, tanto na ol e ul usamos li para criar um item . 

  <!DOCTYPE html>
 <html>
  <body>
   <ol>
        <li> Somar </li>
        <li> Dividir </li>
        <li> Raiz </li>
        <li> Equação </li>
   </ol>
  </body>
 </html>

12. <pre>

Nessa TAG o código fica igual o que está no vscode inclusive contanto espaços, e demais qualidades. 

  <!DOCTYPE html>
 <html>
  <body>
    <pre> As coisas escritas aqui 
    ficão exatamente igual ao jeito 
        que esta aqui no codigo.
    </pre>  
  </body>
 </html>

13. <img src="####">

A tag img é usada pra colocar uma imagem no code html ela não possuí TAG de fechamento pois não é uma TAG de conteúdo em si . 

  <!DOCTYPE html>
 <html>
  <body>
    <img src="foto-zika.png">
  </body>
 </html>
